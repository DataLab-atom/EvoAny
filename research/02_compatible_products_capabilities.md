# OpenClaw 兼容产品现有能力对比

> 调研日期：2026-03-17
> 聚焦：各产品**当前已上线/已有**的能力

---

## 一、QClaw（腾讯 PC 管家团队）

**定位：** 面向普通用户的 OpenClaw 一键安装包，核心差异化：WeChat/QQ 远程控制本地电脑。

### 现有能力
| 能力 | 状态 |
|------|------|
| 一键安装（Windows / macOS） | ✅ 内测中 |
| 零配置接入 WeChat（发消息远程控制电脑） | ✅ |
| 接入 QQ | ✅ |
| 修改表格、发送邮件、操作浏览器（通过 WeChat 指令） | ✅ |
| 直接关联已有 OpenClaw 安装（一键迁移） | ✅ |
| 预集成中国大模型（Kimi、DeepSeek、GLM、MiniMax） | ✅ |
| 完全本地部署（数据不过服务器） | ✅ |
| 免费（beta 期间） | ✅ |

### 局限
- 目前仍处内测，未公开下载
- 功能集是 OpenClaw 的子集（专注 WeChat/QQ 渠道，未暴露全部 Skills）

## 五、其他生态产品现有能力

### PicoClaw（中国工程师开源）
- Go 语言重写 OpenClaw 核心
- 内存：从数百 MB → **<10MB**
- 可在 $10 树莓派 Zero 或微控制器芯片上运行
- 与原版 Skills 格式兼容

### MicroClaw
- **14 个渠道适配器**（含钉钉、QQ、Matrix 等亚洲平台）
- 最广渠道覆盖的 OpenClaw 兼容实现

### IronClaw
- **WASM 沙箱** + 密码学验证（输入/输出/状态转换）
- 面向去中心化应用和零信任环境

### ClawRouter
- OpenClaw 成本路由层
- 根据任务复杂度动态分发给不同 LLM，降低成本

### MCP360 网关
- 统一 MCP 网关，一个接入点覆盖 100+ 工具
- 兼容 Claude、Cursor、YourGPT、n8n 等所有 MCP 客户端
- 自定义 MCP 构建器
